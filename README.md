# AI Resume Builder

A premium, clean resume builder with live preview, template system, ATS scoring, and export helpers.

## Highlights
- Live resume builder + preview
- Templates: Classic / Modern / Minimal
- Accent color themes (5 presets)
- Skills & Projects accordions with tag inputs
- ATS Readiness Score + improvement tips
- Export helpers: Download PDF toast + Copy as Text
- LocalStorage autosave

## Tech Stack
- React
- Vite
- Tailwind CSS

## Local Setup

```bash
npm install
npm run dev
#Build & Preview

npm run preview
